### 📦 安装说明 (Installation)

#### 方法一：从源码构建
1. 克隆仓库：
   ```bash
   git clone https://github.com/archerU/webTab.git
   cd webTab
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 构建扩展：
   ```bash
   npm run build
   ```

4. 在 Chrome 中加载：
   - 打开 `chrome://extensions/`
   - 启用"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择 `dist` 文件夹

#### 方法二：使用打包文件
1. 下载 `webTab-extension.zip`
2. 解压文件
3. 在 Chrome 中加载解压后的文件夹

### 📝 使用说明 (Usage)

1. **添加快捷方式**：
   - 点击分类中的 "+" 按钮
   - 填写名称和 URL
   - 选择图标模式（自动/上传/颜色）
   - 点击"添加快捷方式"

2. **管理分类**：
   - 点击右上角设置按钮
   - 在"分类"部分添加或删除分类

3. **切换语言**：
   - 点击顶部右侧的地球图标按钮
   - 或在设置中选择语言

4. **打开新标签页**：
   - 点击扩展图标即可打开新标签页
