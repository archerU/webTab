# webTab - 自定义网站导航新标签页

## 简介

webTab 是一款功能强大的 Chrome 浏览器新标签页扩展，让您能够完全自定义您的网站导航体验。告别默认的空白标签页，创建一个属于您自己的个性化导航中心。

## 核心功能

### 🎯 自定义网站导航

- **快速访问常用网站**：将您最常访问的网站添加到新标签页，一键直达
- **智能分类管理**：创建多个分类来组织您的网站，如"工作"、"娱乐"、"学习"等
- **拖拽排序**：通过简单的拖拽操作，自由调整分类和快捷方式的顺序
- **跨分类移动**：轻松将快捷方式从一个分类移动到另一个分类

### 🎨 个性化定制

- **自定义图标**：
  - **自动获取**：自动从网站获取 favicon 图标
  - **上传图标**：支持上传自定义图片作为图标（最大 2MB）
  - **颜色选择**：选择纯色背景，显示网站名称首字母
- **背景图片**：设置您喜欢的背景图片，打造独特的视觉体验
- **毛玻璃效果**：精美的毛玻璃设计，让界面更加现代化和美观

### 🌍 国际化支持

- **双语界面**：完整支持中文和英文
- **自动语言检测**：根据浏览器语言自动设置界面语言
- **快速切换**：一键切换语言，无需刷新页面

### 💾 数据管理

- **本地存储**：所有数据安全存储在您的设备本地
- **Google 账号同步**：登录 Google 账号后，数据自动同步到云端，多设备无缝使用
- **导入/导出配置**：
  - 导出配置为 JSON 文件，随时备份
  - 导入之前的备份，快速恢复所有设置
- **数据安全**：所有数据仅存储在本地或您的 Google 账号中，不会上传到任何第三方服务器

### ⚡ 便捷操作

- **快速添加**：点击"+"按钮即可快速添加快捷方式
- **批量管理**：在设置页面统一管理所有分类和快捷方式
- **编辑分类**：随时修改分类名称，保持导航整洁
- **删除功能**：轻松删除不需要的分类或快捷方式

## 使用场景

### 工作场景
- 创建"工作工具"分类，包含常用办公网站
- 快速访问项目管理工具、文档协作平台
- 提高工作效率，减少查找网站的时间

### 学习场景
- 创建"学习资源"分类，整理学习网站
- 快速访问在线课程、文档资料
- 保持学习环境的整洁和专注

### 娱乐场景
- 创建"娱乐"分类，包含视频、音乐、游戏网站
- 一键访问您喜欢的娱乐平台
- 个性化定制，让浏览更愉悦

## 技术特点

- **轻量级**：占用资源少，启动速度快
- **响应式设计**：适配各种屏幕尺寸
- **隐私保护**：所有数据本地存储，不上传服务器
- **离线可用**：即使没有网络，也能正常使用本地数据

## 安装使用

1. 从 Chrome Web Store 安装扩展
2. 打开新标签页，开始自定义您的导航
3. 点击"+"按钮添加快捷方式
4. 在设置中管理分类和配置
